# pragya's portfolio

personal portfolio site — built with HTML, CSS, and vanilla JS.

## structure
```
pragya-portfolio/
├── index.html        # main site (single file, self-contained)
└── README.md
```

## deploying
hosted on vercel. any push to main auto-redeploys.

## updating
1. edit `index.html`
2. commit + push to github
3. vercel deploys automatically in ~30s
